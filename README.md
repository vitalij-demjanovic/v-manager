# Project V-Manager App

## Introduction
Project Management App is a web application designed to help teams and individuals manage their projects efficiently. With features like task tracking, project deadlines, and role-based access control, this app provides a comprehensive solution for all your project management needs.

## Features
- **Project Management**: Create, edit, and manage multiple projects.
- **Task Management**: Assign tasks to team members, set priorities, and track progress.
- **User Roles**: Differentiate access levels with role-based permissions.
- **Real-time Updates**: Stay up-to-date with real-time notifications and updates.
- **Responsive Design**: Access your projects on any device, anytime.

## Tech Stack
- **Frontend**: React, Next.js, Tailwind CSS, shadcn ui
- **Backend**: Supabase (PostgreSQL), Next.js Server Actions
- **Authentication**: Supabase Auth
  - **Deployment**: Vercel
